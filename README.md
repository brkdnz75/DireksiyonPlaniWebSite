# DireksiyonPlanı - Sürücü Kursu Yönetim Sistemi

Modern, sade ve profesyonel bir web sitesi. Sürücü kursları için geliştirilmiş rezervasyon ve yönetim yazılımının tanıtımı.

## Özellikler

- Modern React + TypeScript + Vite
- Tailwind CSS ile responsive tasarım
- React Router ile çoklu sayfa desteği
- Lucide React ikonları
- Mobil uyumlu tasarım

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Sayfalar

- **Ana Sayfa**: Hero section ve özellik özeti
- **Özellikler**: Detaylı özellik açıklamaları
- **Nasıl Çalışır**: 3 adımlı kullanım süreci
- **Video Eğitimler**: YouTube video gömme alanı
- **Sosyal Medya**: Instagram ve YouTube bağlantıları
- **İletişim**: Demo talep formu

## Teknolojiler

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React
